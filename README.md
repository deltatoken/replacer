# replacer
Replacer tool:
Replace each occurens by regexes in all files of a specific folder

### Install:

```
$ git clone https://github.com/gymad/replacer.git
$ cd replacer
```

### Config: 

Follow the instruction in `config.php` or `config.example.php` to set regexes and folder etc..

### Run replacer:

```
$ sudo php index.php
```
